    A few basic things

    <Esc> is your friend (puts you back in Normal mode)

    . is magic (repeats last edit)

    i enter Insert mode

    <Cr> is vim talk for the Enter/Return key

    <C-r> is vim talk for Ctrl + R (Redo)

    Save current file: :w<Cr>

    Save to a filename (Save As...): :w filename<Cr>

    Exit (all changes are saved): :q<Cr>

    Exit without saving (ignore changes): :q!<Cr>

    Save and Exit: :wq<Cr> or ZZ (note both Zs are upper case)

    ~/.vimrc is the default location of your configuration file (~ is current user's home folder)
